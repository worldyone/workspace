Usage
> acc new abc123
... choice problem
> cd abc123/a
... solve problem in main.py
> acc s # submitの略. フォルダ名から提出先URLを推測して提出する
> cd ..
> acc add
... try next problem

1. VS Code上でCtrl + Shift + Bでサンプルテストケースでのテストを自動実行
2. VS Code上でF5で手入力値(input.txt)でのテスト実行

# atcoder-cliの設定フォルダ
cd `acc config-dir`
